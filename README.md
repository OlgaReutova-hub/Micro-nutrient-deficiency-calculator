# Калькулятор поступления микронутриентов на низкокалорийной диете

Веб-приложение на [Streamlit](https://streamlit.io) для подбора типового суточного меню (1200–2000 ккал) и оценки поступления макро- и микронутриентов относительно суточных норм.

## Возможности

- Выбор пола, калорийности рациона (1200 / 1500 / 1800 / 2000 ккал) и типа меню:
  - домашнее;
  - офисное;
  - перекус в городе.
- Отображение меню по приёмам пищи (завтрак, обед, перекус, ужин).
- Расчёт поступления нутриентов с цветовой индикацией:
  - зелёный — ≥ 90% от нормы;
  - жёлтый — 60–89%;
  - красный — &lt; 60%.

## Требования

- Python 3.10+

## Установка и запуск локально

```bash
git clone <URL-вашего-репозитория>
cd <имя-репозитория>

python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate

pip install -r requirements.txt
streamlit run app.py
```

Приложение откроется в браузере по адресу: **http://localhost:8501**

## Деплой на Streamlit Community Cloud

1. Загрузите содержимое этой папки в репозиторий GitHub.
2. На [share.streamlit.io](https://share.streamlit.io) создайте новое приложение.
3. Укажите:
   - **Repository** — ваш репозиторий;
   - **Branch** — `main` (или нужная ветка);
   - **Main file path** — `app.py`.

Файл `requirements.txt` и каталог `.streamlit/` подхватятся автоматически.

## Данные и методология

Значения поступления нутриентов — **демонстрационные усреднённые оценки** по типовым блюдам; не предназначены для медицинской диагностики.

Нормы потребления микро- и макронутриентов основаны на методических рекомендациях **MP 2.3.1.0253-21** «Нормы физиологических потребностей в энергии и пищевых веществах для различных групп населения Российской Федерации» (утв. Роспотребнадзором 22.07.2021).

## Вспомогательные скрипты

Из корня репозитория:

```bash
# Экспорт меню в Markdown (результат: scripts/menu_1200_2000_menus.md)
python scripts/export_menus_md.py
```

Скрипт `generate_extended_menu_json.py` пересобирает `data/menu_1200_2000_extended_nutrients.json` из исходного файла `menu 1200 - 2000.json` (если он есть в каталоге `scripts/`).
